SkillSource

A fully resolved skill source.

Fields

NameTypeDescription
sourceStringThe raw source string (URL, path, or owner/repo shorthand).
pathOption<String>Subdirectory path within the source (for multi-skill repos).
revOption<String>Pinned revision (git commit, tag, or branch).
versionOption<String>Required SKILL.md version.
kindSkillSourceKindSource-type-specific data.
